# Internet of Things (IoT) Applications in Smart Homes

## 📌 Project Overview
This project is a research report on **Internet of Things (IoT) Applications in Smart Homes**, created as **Task 1** for the **CodeAlpha IoT Internship**.

The report explains how IoT technology enables smart homes by connecting devices through the internet to improve convenience, security, energy efficiency, and automation.
